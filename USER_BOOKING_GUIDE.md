# دليل الاستخدام السريع - نظام الحجز

## كيفية استخدام النظام

### 1. البحث عن رحلة
1. افتح صفحة Dashboard
2. املأ نموذج البحث:
   - اختر مدينة المغادرة
   - اختر مدينة الوصول
   - اختر تاريخ السفر
   - اختر نوع الرحلة (ذهاب/عودة/ذهاب وعودة)
   - حدد عدد المسافرين (1-8)
3. اضغط "بحث عن رحلات"

### 2. اختيار السيارة
1. ستظهر لك قائمة بالسيارات المتاحة
2. كل سيارة تعرض:
   - اسم السيارة ورقم اللوحة
   - عدد المقاعد الكلي
   - عدد المقاعد المحجوزة
   - عدد المقاعد المتاحة
3. اضغط "احجز الآن" على السيارة المناسبة

### 3. اختيار المقاعد
1. ستفتح صفحة اختيار المقاعد
2. المقاعد ملونة حسب الحالة:
   - **رمادي**: متاح للحجز
   - **أخضر**: مقعد اخترته
   - **أحمر**: محجوز بالفعل
3. اضغط على المقاعد التي تريدها (حسب عدد المسافرين)
4. املأ بياناتك:
   - الاسم
   - رقم الهاتف
5. اضغط "تأكيد الحجز"

### 4. تأكيد الحجز
- إذا نجح الحجز: ستظهر رسالة نجاح وسيتم توجيهك للصفحة الرئيسية
- إذا فشل الحجز: ستظهر رسالة خطأ (مثلاً إذا حجز شخص آخر نفس المقعد)

## الميزات الأمنية

### 1. منع التداخل
- لا يمكن لشخصين حجز نفس المقعد في نفس الوقت
- يتم التحقق من توفر المقاعد قبل إتمام الحجز
- استخدام Database Transactions لضمان عدم التداخل

### 2. التحقق من البيانات
- جميع البيانات يتم التحقق منها قبل القبول
- رسائل خطأ واضحة عند وجود مشكلة

### 3. تحديث فوري
- إذا حجز شخص مقعد أثناء اختيارك، سيتم تحديث الصفحة
- المقاعد المحجوزة تظهر باللون الأحمر فوراً

## API Endpoints

### للمطورين

#### 1. جلب معلومات المقاعد
```http
POST /bookings/bus-seats
Content-Type: application/json

{
  "schedule_trip_id": 1,
  "bus_id": 1
}
```

#### 2. التحقق من توفر المقاعد
```http
POST /bookings/check-seats
Content-Type: application/json

{
  "schedule_trip_id": 1,
  "bus_id": 1,
  "seat_numbers": [12, 13, 14]
}
```

#### 3. إنشاء حجز
```http
POST /bookings/create
Content-Type: application/json

{
  "customer_name": "أحمد محمد",
  "phone_number": "01234567890",
  "subtotal_price": 300,
  "items": [
    {
      "schedule_trip_id": 1,
      "bus_id": 1,
      "type": "go_way",
      "seat_numbers": [12, 13, 14],
      "price": 300,
      "from_city": "القاهرة",
      "to_city": "الإسكندرية"
    }
  ]
}
```

## حالات الحجز

1. **pending**: في انتظار التأكيد/الدفع
2. **confirmed**: مؤكد
3. **cancelled**: ملغي

ملاحظة: فقط الحجوزات بحالة `pending` أو `confirmed` تحجز المقاعد.

## أسئلة شائعة

### س: ماذا يحدث إذا حاول شخصان حجز نفس المقعد؟
ج: سيتم قبول الحجز الأول فقط. الشخص الثاني سيحصل على رسالة خطأ وسيتم تحديث المقاعد المتاحة.

### س: كم عدد المقاعد التي يمكنني حجزها؟
ج: يجب أن يكون عدد المقاعد مساوياً لعدد المسافرين الذي اخترته في البحث (1-8).

### س: هل يمكنني تغيير المقاعد بعد اختيارها؟
ج: نعم، قبل الضغط على "تأكيد الحجز" يمكنك النقر على المقعد مرة أخرى لإلغاء اختياره.

### س: ماذا لو لم أجد مقاعد متاحة؟
ج: يمكنك:
1. اختيار سيارة أخرى من نفس الرحلة
2. البحث عن تاريخ آخر
3. تقليل عدد المسافرين

## الدعم الفني

إذا واجهت أي مشكلة:
1. تحقق من رسالة الخطأ المعروضة
2. أعد تحميل الصفحة
3. جرب المتصفح في وضع التصفح الخفي
4. تواصل مع الدعم الفني
